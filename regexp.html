Quantifiers:

matches
n+ any string with atleast one n
n* any string with zero or more n
n? any string with one or less n
n{x} any string with a sequence of x ns
n$ any string with n at the end
^n any string with n at the beginning
?=n any string followed by string n
?!n any string not followed by string n

Brackets

Expression Discription Examples
[abc] Find any character between the brackets, i.e. abc
[^abc] Find any character NOT between the brackets, i.e. d-z
[123] Find any digit between the brackets, i.e. 123
[^123] Find any digit NOT between the bracket, i.e. 4567890
